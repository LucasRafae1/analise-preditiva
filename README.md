# Análise Preditiva de Dados Ambientais

Análise preditiva completa utilizando dados reais de Emergia (Emergy Analysis),
metodologia científica que mede o custo ambiental de produção de recursos.

## 🔍 Métodos Utilizados
- Razão K
- Correlação de Pearson
- Regressão Linear e Log-Log
- Machine Learning

## 🛠️ Tecnologias
- Python
- Google Colab
- Excel 

## 📁 Arquivos
- `EmergyDatabase.xlsx` — Base de dados original
- `analise_uev.xlsx` — Planilha com análise completa
- `analise_emergy_COLAB.ipynb` — Notebook com o código e resultados
